# Exemplos de Títulos de Quests - MindQuest

**Versão:** 1.0  
**Data:** 2025-12-22

## Princípios

| ❌ EVITAR | ✅ FAZER |
|-----------|----------|
| Termos técnicos (TCC, Mindfulness, 4-7-8) | Linguagem do usuário |
| Títulos genéricos de app de tarefas | AÇÃO + RESULTADO específico |
| Foco na técnica | Foco no benefício/transformação |
| "Pratique X", "Faça exercício de Y" | Verbos de impacto: Blinde, Instale, Desative |
| Menção explícita a sabotadores | Foco na ação/benefício |

---

## Fórmula do Título

```
[VERBO DE IMPACTO] + [OBJETO] + [RESULTADO/CONTEXTO]
```

**Verbos de impacto:** Blinde, Instale, Desative, Elimine, Desbloqueie, Ative, Reconfigure, Resete, Mapeie, Transforme, Acelere, Proteja, Recupere, Libere, Conquiste

## Regras para os Agentes

### System Prompt - Diretrizes de Título

```
REGRAS PARA TÍTULO DE QUEST:

1. NUNCA usar termos técnicos: TCC, Mindfulness, Estoicismo, 4-7-8, Grounding, Reframing
2. NUNCA mencionar nome de sabotador: "Desafie o Sr. X", "Seu Hiper-Realizador"
3. SEMPRE usar fórmula: VERBO DE IMPACTO + OBJETO + RESULTADO
4. SEMPRE conectar ao contexto da conversa (objetivo, emoção, situação)
5. MÁXIMO 80 caracteres
6. Linguagem direta, sem rodeios

VERBOS APROVADOS:
Blinde, Instale, Desative, Elimine, Desbloqueie, Ative, Reconfigure, Resete, 
Mapeie, Transforme, Acelere, Proteja, Recupere, Libere, Conquiste, Defina,
Liste, Escolha, Documente, Crie, Teste, Delegue, Substitua, Registre

EXEMPLOS:
✅ "Proteja seu foco das distrações internas"
✅ "Defina a única feature que valida seu MVP"
✅ "Recupere 5 minutos de clareza mental"
❌ "Pratique Mindfulness para ansiedade"
❌ "Desafie o Sr. Inquieto com pausas"
❌ "Técnica de Reestruturação Cognitiva"
```

## Validação de Título

Antes de gravar, validar:

- [ ] Tem verbo de impacto no início?
- [ ] Conecta ao contexto do usuário?
- [ ] Evita termos técnicos?
- [ ] Evita nome de sabotador?
- [ ] Tem menos de 80 caracteres?
- [ ] É ação rápida (< 10 min)?
- [ ] Foca em resultado, não em técnica?
